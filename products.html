РЕХАУ INTELIO 80
Область применения и решаемые проблемы

Окна из профилей INTELIO станут прекрасным выбором для:

  • Домов с минимальным потреблением энергии
  • Реконструируемых объектов
  • Строительства зданий класса “премиум”

Система INTELIO используется:

  • Для изготовления окон
  • Для балконных дверей
  • Для входной группы

Профиль дает возможность выполнить окна в эксклюзивном дизайне с использованием широчайшего выбора цветов и декора

  • Домов с минимальным потреблением энергии
  • Реконструируемых объектов
  • Строительства зданий класса “премиум”

Система INTELIO используется:

  • Для изготовления окон
  • Для балконных дверей
  • Для входной группы

Профиль дает возможность выполнить окна в эксклюзивном дизайне с использованием широчайшего выбора цветов и декора
